<br/></br>### 课程介绍:
本课程的主要内容包括计算机技术发展过程及趋势、计算机系统组成、数据的表示与运算、计算机硬件和软件、操作系统、办公软件、程序设计语言、数据库技术、多媒体技术、计算机网络、网页设计和信息安全。通过本课程的学习，不仅可以掌握计算机的基本理论和基础知识，而且能初步具备利用计算机分析问题、解决问题的意识与能力，提高计算机素养，为将来应用计算机的相关知识和技术解决所学专业领域的实际问题打下良好的基础。
